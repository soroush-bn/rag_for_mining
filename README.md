# rag_for_mining

Instead of reading hundreds of pages of mining safety regulations and legislation across Newfoundland and Labrador, ask our RAG. You can upload images, documents, or simply chat with it.

Here is the website if you'd like to try it yourself:

https://rag-for-mining.vercel.app/
